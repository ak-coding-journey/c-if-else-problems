# c-if-else-problems
A collection of simple C programs for beginners. Each program demonstrates basic conditional logic and number properties:  1:Leap year check .
2:Divisibility by 5 .
3:Divisibility by 7 .
4:Armstrong number check  .
5:Triangle validity .

Features
1-Clear input/output prompts
2-Easy-to-read code structure
3-Practical examples of if-else statements

Installation & Run:-
git clone https://github.com/ak-coding-journey/c-if-else-problems.git
cd c-if-else-problems

Compile the program using gcc:-
gcc program.c -o program

Run the executable:-
./program

Example:-

Enter any year: 2024
2024 is a Leap Year.

Enter any integer to check divisibility by 5: 25
25 is divisible by 5.

Enter any integer to check divisibility by 7: 14
14 is divisible by 7.

Enter a 3-digit number: 153
153 is an Armstrong number.

Enter three sides of the triangle: 3 4 5
It is a valid triangle.
